--------------------------------------------------------------------------------

				ソンタクロースの軌跡

--------------------------------------------------------------------------------

1.はじめに
　この度は、「ソンタクロースの軌跡」をダウンロードしていただきありがとうございます。
　本書では、おもにこのゲームの概要、操作方法について説明しております。

2.概要
　子どもたちが欲しがっているプレゼントを忖度して、
　サンタクロースを操作して、プレゼントを配りましょう。
　家から出る吹き出しを見てプレゼントを選び、
　投下して家に当たれば得点になります。

3.操作方法
　← → : サンタの移動
　↑ ↓ : 投下するプレゼントの選択
　SPACE : プレゼント投下

4.注意
　上級では、サンタ狩りを実行している人々が、ときどき大砲をぶっ放します。
　当たったらその時点でゲーム終了となりますので、
　左右キーを押して弾を避けてください。
　
　また、20秒ごとに加速します。
　速度が遅いうちに点を稼ぎましょう。
